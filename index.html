The problem

Most independent groups have never seen the other side's numbers.

Rates for the same procedure, in the same metro, routinely differ by 2× or more between payers — and between practices on the same payer. A practice that has never benchmarked can't know which end of that spread it's on. Payers know. Now the data to level that asymmetry is public — if you can process multi-gigabyte disclosure files.

The data

Built only from public, federally mandated disclosures.

  • Payer Transparency-in-Coverage files: negotiated professional rates, per practice, per code
  • Hospital price-transparency files (2026 fields): what payers actually paid, from remittance data
  • CMS Physician Fee Schedule: the CY2026 Dallas-locality Medicare baseline
  • No claims data, no patient data — nothing that touches PHI

Methodology

Professional-billing negotiated rates from payer in-network files (flagship commercial network per payer), restricted to gastroenterology-taxonomy NPIs practicing in DFW-metro counties via the NPPES registry, deduplicated by tax ID and rate, and reported as percentiles per code and payer. Medicare comparisons use the final CY2026 Physician Fee Schedule, Dallas locality, matched to the place of service each rate is published for.
